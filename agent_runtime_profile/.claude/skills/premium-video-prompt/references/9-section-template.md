# 参考生视频成品提示词模板

本模板是 `premium-video-prompt` skill 的主输出结构。生成 reference_video unit prompt 时，必须优先套用本模板，不要自行发明新的板块顺序。

## 写作规则

- 图片引用声明必须放在最前面。
- 图片编号必须连续：`图片1`、`图片2`、`图片3`、`图片4`。
- 图片编号必须与实际传入的 `references` 顺序一致。
- 正文只使用干净的 `图片1` / `图片2` 写法，禁止出现 `I图片1`、`!图片2`、`#图片1`、`2图片4` 等污染符号。
- 风格只保留一个明确方向，不要写互相冲突的风格，例如“中国古代/现代风格”。
- 每个切片段都要有：画面、运镜、对白或音效；没有对白时明确写“无对白”。
- 0s-0.15s 作为废帧缓冲，不安排关键动作、关键表情和关键对白。
- 禁止字幕、水印、Logo、画面文字、BGM。

## 标准模板

```text
【图片引用声明】
图片1：角色“{character_name_1}”的外观参考。
图片2：角色“{character_name_2}”的外观参考。
图片3：场景“{scene_name}”的环境参考。
图片4：道具“{prop_name}”的外观参考。

【全局视频要求】
{visual_style}。
竖屏，多场景，多角度，60fps流畅动画。
有环境音和动作音效，无音乐，无字幕，不出现任何文字、水印、Logo。
中文对白，人物动作流畅，运镜丝滑。
0s-0.15s 作为废帧缓冲，不安排关键动作、关键表情和关键对白。

【场景设计】
场景：图片3，{time_of_day}，{interior_or_exterior}，{lighting_mood}。
{set_dressing}
整体空间氛围：{space_atmosphere}

【目标情绪】
{emotion_arc}

【片段说明】
片段1：{slice_1_goal}
片段2：{slice_2_goal}
片段3：{slice_3_goal}
片段4：{slice_4_goal}

【切片段1】
画面：{slice_1_visual_action}
运镜：{slice_1_camera_direction}
对白：{slice_1_dialogue_or_none}
配音/音效：{slice_1_voice_or_sfx}

【切片段2】
画面：{slice_2_visual_action}
运镜：{slice_2_camera_direction}
对白：{slice_2_dialogue_or_none}
配音/音效：{slice_2_voice_or_sfx}

【切片段3】
画面：{slice_3_visual_action}
运镜：{slice_3_camera_direction}
对白：{slice_3_dialogue_or_none}
配音/音效：{slice_3_voice_or_sfx}

【切片段4】
画面：{slice_4_visual_action}
运镜：{slice_4_camera_direction}
对白：{slice_4_dialogue_or_none}
配音/音效：{slice_4_voice_or_sfx}

【负面约束】
禁止出现：字幕、文字、水印、Logo、BGM、画面边框。

禁止出现：镜头跳帧、剧烈抖动、动作僵硬、口型与对白明显不同步。
禁止出现：过曝、过饱和、塑料质感皮肤、背景细节融化。
```

## 示例

```text
【图片引用声明】
图片1：角色“萧近宸”的外观参考。
图片2：角色“侍卫”的外观参考。
图片3：场景“王府书房”的环境参考。
图片4：道具“《冷面王爷的娇软小逃妻》话本”的外观参考。

【全局视频要求】
3D动画CG建模风格，中国古代王府氛围。
竖屏，多场景，多角度，60fps流畅动画。
有环境音和动作音效，无音乐，无字幕，不出现任何文字、水印、Logo。
中文对白，人物动作流畅，运镜丝滑。
0s-0.15s 作为废帧缓冲，不安排关键动作、关键表情和关键对白。

【场景设计】
场景：图片3，王府书房，白天内景，光线昏暗压抑。
深色紫檀木书案上摆放笔墨纸砚，正中央放着图片4话本。
整体空间宽敞威严，突出萧近宸冷酷、压迫的气场。

【目标情绪】
情绪加压：禀报 → 冷眼审视 → 紧张补充 → 暴怒砸书。

【片段说明】
片段1：图片2禀报行踪。
片段2：图片1冷眼审视。
片段3：图片2紧张补充。
片段4：图片1暴怒砸书。

【切片段1】
画面：图片2单膝跪地，低着头，神色恭敬中带着一丝不自然。图片1坐在书案后，修长手指有节奏地敲击图片4。
运镜：过肩机位，胸平高度，从图片1敲击话本的手，移动到图片2单膝跪地的中景。速度平缓，焦点落在图片2略显尴尬的神情。
对白：图片2：“殿下，属下查了那间酒馆，其幕后东家隐蔽，江小姐近半月出入频繁，行踪诡秘。”
配音/音效：青年男性，低沉有力，语气恭敬，略带尴尬；保留衣料摩擦声与书房轻微环境声。

【切片段2】
画面：图片1敲击话本的手指蓦然停顿。他微微抬起下巴，眼神如寒潭般沉冷，充满压迫感地盯着图片2。
运镜：正面机位，眼平高度，从手指停顿特写切到图片1沉冷双眼近景。速度偏快，焦点落在他的审视目光。
对白：图片1：“还有呢？”
配音/音效：青年男性，低沉冷冽，语气平淡但有压迫感；手指停顿瞬间保留极轻的木桌敲击尾音。

【切片段3】
画面：图片2头埋得更低，额头渗出冷汗，喉结紧张滚动，眼神闪躲，声音越来越小。
运镜：侧俯拍，腰平高度，从图片2喉结特写移动到他闪躲的眼睛近景。速度缓慢，焦点落在图片2极度紧张的表情。
对白：图片2：“还有……那话本，似乎也与江小姐有关。”
配音/音效：青年男性，语气更低更虚，带明显迟疑；加入轻微吞咽声和压低的呼吸声。

【切片段4】
画面：图片1眼神骤冷，猛地抓起图片4，重重砸在书案上。纸页震动，笔架轻晃，图片2身体一僵。
运镜：从图片4被抓起的特写快速拉到图片1暴怒的中近景，再切到图片2受惊僵住的反应。速度紧凑有力，落点在图片1压抑怒意的脸。
对白：图片1：“荒唐。”
配音/音效：青年男性，低沉压怒，字音短促；加入话本砸桌的沉闷撞击声、纸页震颤声、笔架轻响。

【负面约束】
禁止出现：字幕、文字、水印、Logo、BGM、画面边框。
禁止出现：角色脸部崩坏、手指畸形、肢体变形、服装颜色严重偏离参考图。
禁止出现：镜头跳帧、剧烈抖动、动作僵硬、口型与对白明显不同步。
禁止出现：过曝、过饱和、塑料质感皮肤、背景细节融化。
```
